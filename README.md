# Project112025Mia

This repository provides a ready-to-use **RStudio Server Docker environment** with the custom R package [**CompuDataAnalysis**](https://github.com/miacarboni-creator/CompuDataAnalysis) pre-installed. You can also pull the pre-built Docker image from [Docker Hub](https://hub.docker.com/repository/docker/miacarbonicreator/project112025mia/general).

## Repository Contents

- **Docker Files**: ZIP folder with all files to build the Docker container locally.  
- **Docker Guide**: HTML file with a step-by-step tutorial on how to create the Docker container.  
- **Analysis Report**: HTML file containing the markdown of analyses performed on the data included in the ZIP folder.  
- **R Package Guide**: HTML file explaining how to create the **CompuDataAnalysis** R package.  

## Features

- Ready-to-use **RStudio Server** environment with R 4.4.1.  
- Includes a **data folder** with sample data for analysis.  
- Pre-installed **CompuDataAnalysis** R package for:  
  - High-throughput data manipulation and aggregation  
  - Benchmarking and performance evaluation  
  - Working with RNA-seq data, laboratory measurements, genomic peaks/SNPs, and cell type integration
